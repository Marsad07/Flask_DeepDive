Help & Support

Welcome to the help section. If you're unsure how to use any part of the app, this guide will walk you through it.

📌 Getting Started

Use the sidebar on the left to navigate between subjects, tasks, and tools.

📚 Subjects

  • View all your subjects from the sidebar.
  • Click a subject to see its tasks.
  • Use the edit icon to update a subject.

📝 Tasks

  • Add tasks inside a subject page.
  • Mark tasks as completed using the check button.
  • Delete tasks using the delete button.

⏱ Pomodoro Timer

Use the Pomodoro button to start a focused study session.

🗓️ Study Plans

What are Study Plans?

Study Plans help you organize your tasks into a weekly timetable, showing you exactly when to study each subject.

How to Create a Study Plan:

  • Click "Study Plans" in the sidebar
  • Click "+ Create study plan"
  • Give your plan a title (e.g., "Exam Week Schedule")
  • Click "+ Add Task" to add tasks to your schedule

For each task, select:

  • Which task to schedule
  • Which day (Mon-Fri)
  • Start Time
  • End Time

Then Click "Create Study Plan" when done
